Pixel Privacy Project

Using image enhancement technology to protect user privacy

About

The pixel privacy project develops image enhancement technology that helps to protect user privacy. Specifically, it is dedicated to creating new approaches that invisibly change or visibly enhance images in such a way that it is no longer possible to automatically extract privacy-sensitive information. The methods proposed, PIRE and PERC, are two algorithms that change the structure of the pixels which creates a new image. This new image won't differ semantically in the human eye, but cannot be used for purposes users did not intend on when posting the picture online. By doing so protecting the users privacy

What do we battle?

In this era of social media, a lot of people post pictures online. Their intentions is to share their life with friends and followers. These images however can be used by content retrieval computersystems to extract data the user did not intent to share. Think of data on security systems, locations, what colour clothing one wears frequently. How does that work? Content based information retrieval systems (CBIR) accept images as an input and ask a question similar to: what do you see on this picture? The system returns a list of relevant and similar images as an answer to that question. The input image is represented in a vector that is based on the content of an image. You can consider this vector a numerical “fingerprint” of what is displayed on the image. A CBIR system then compares that vector to content based vectors of other images in a database and selects the most similar vectors as images that have something similar or something to do with the original image. An example: you post a picture of yourself in front of the Stevens Church in Nijmegen, wearing a red dress at noon. The CBIR takes that as input, asks the question: what do you see on this picture. It compares the fingerprints of the picture with images in its database, returning a list of pictures from the Sint Stevens church, Nijmegen, red dresses and lunchtime. Before you know it, the system shows you advertisements of lunchrooms in Nijmegen close to the Sint Stevens church and advices you to wear a red dress to another meeting. Not really what you expected from posting the "innocent" instagram picture.

The website

On this website you can find examples of the beautiful images we created using image enhancement techniques that also assure user privacy. You can wander around the gallery and be amazed by the beautiful pictures. If you got curious to how we did it, you can have a look at the algorithms page. We described the mechanisms behind each filter using understandable language. And if you really want to dive deep and explore the technical details, you can find links to the published papers on each algorithm and the github repository where you can study the code. In the contact section you can find who to contact if you have questions, want a collaboration or want to have a nice chat on this topic!

Algorithms

This section contains overviews and links to the algorithms page containing further info about algorithms used in this project.

PIRE

PIRE changes the images "fingerprint" so a CBIR cannot use that fingerprint to find similar images. Click to find out more!

PERC

PerC alters images with respect to the perceptual colour so the image looks the same to humans but different to a CBIR. Click here to find out more!
